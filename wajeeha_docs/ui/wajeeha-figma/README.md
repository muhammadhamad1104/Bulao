# Bulao UI/UX Design Handoff

Designer: Wajeeha Kamran  
Role: UI/UX Designer  
Project: Bulao — “bolo, aur kaam ho jaye”

## Overview

This folder contains the exported UI/UX design screens created in Figma for the Bulao mobile application. These screens were used as the visual reference for the Flutter frontend implementation.

## Included Screens

1. Splash Screen
2. Login Screen
3. Sign Up Screen
4. Forgot Password Screen
5. Home Screen
6. Processing Loading Screen
7. Processing Agents Screen
8. Provider Selection Screen
9. Booking Confirmed Screen
10. Live Tracking Screen
11. Feedback Screen

## Design Style

The Bulao UI follows a soft premium glass/neumorphic style with:
- Cream/off-white background
- Navy blue and golden accent colors
- Rounded cards and soft shadows
- Voice-first interaction flow
- Clean service booking experience

## Fonts Used

- IBM Plex Sans Condensed for main headings
- IBM Plex Sans / Inter for body text and descriptions
